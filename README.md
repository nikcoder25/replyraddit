# ReplyRaddit

🔗 **Live site:** https://nikcoder25.github.io/replyraddit/

Your AI co-pilot for **brand-safe Reddit marketing**. ReplyRaddit finds the
conversations where your customers already hang out, scores them by relevance,
and drafts authentic, value-first replies you review and post — so you show up
helpful, not spammy.

> This is a marketing landing page (a tribute build inspired by the
> ReplyDaddy concept). Copy, design, and assets are original. Not affiliated
> with or endorsed by Reddit, Inc.

## What's inside

A fast, dependency-free, fully responsive landing page:

- **Hero** with product mock-up (opportunity score + suggested reply draft)
- **Problem** section
- **How it works** — 4 steps (Brand analysis → Persona setup → Smart discovery → Reply generation)
- **Features** grid (relevance scoring, AI drafts, keyword tracking, rule awareness, tone controls, dashboard)
- **Brand-safe** section (no account linking, no auto-posting, human-reviewed)
- **Testimonials**
- **Pricing** — Solopreneur $49/mo · Growing Business $199/mo · Agency/Team $799/mo (+ lifetime note)
- **FAQ**
- **Final CTA** + footer

## Tech

Plain HTML, CSS, and a small vanilla-JS file. No build step.

| File | Purpose |
|------|---------|
| `index.html` | Page markup |
| `styles.css` | Styling, layout, responsive rules |
| `script.js`  | Mobile nav, footer year, scroll reveal |

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy

Static — host anywhere (Netlify, Vercel, GitHub Pages, Cloudflare Pages).
Point the host at the repo root; no build command needed.
